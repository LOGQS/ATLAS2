# ATLAS2 - Advanced AI Assistant

ATLAS2 is a sophisticated, feature-rich AI chat application designed as a powerful, multimodal assistant. It goes far beyond a simple chatbot, integrating advanced features for developers and power users.

## 🚀 Key Features

- **Multi-Provider AI Integration** - Support for Google Gemini, OpenRouter, and Groq models
- **Multimodal Conversations** - Upload and discuss images, documents, audio, and video files
- **Creations System** - Generate and manage code, diagrams (Mermaid), HTML, React components, and more
- **Knowledge Base Profiles** - Create specialized workspaces with document collections for context-aware conversations
- **Task Management** - Structured multi-step task planning and execution
- **Real-time Streaming** - Live response generation with background processing
- **File Management** - Advanced file handling with PDF viewer, image annotation, and more
- **Speech Integration** - Text-to-Speech and Speech-to-Text capabilities

## 🛠 Tech Stack

**Backend:**
- Python/Flask with WebSocket support
- Google Gemini API, OpenRouter, Groq integration
- Faster-Whisper for speech processing
- JSON-based data persistence

**Frontend:**
- React/TypeScript
- Real-time streaming with Server-Sent Events
- Syntax highlighting, PDF viewing, code sandboxing
- Modern dark-themed UI

## 📋 Core Systems

1. **Chat Session Management** - Organized conversation history
2. **Profile Management** - Knowledge bases with smart document retrieval
3. **File Handling** - Secure upload and processing pipeline
4. **Creations System** - Structured content generation and display
5. **Message Handling** - Real-time streaming communication
6. **Task System** - Complex multi-step goal management
7. **Unified AI Integration** - Seamless multi-provider support

## 🔧 Quick Setup

1. Clone the repository
2. Install Python dependencies: `pip install -r requirements.txt`
3. Install Node.js dependencies: `npm install` in frontend dir
4. Configure API keys in environment variables
5. Start concurrently: npm start
